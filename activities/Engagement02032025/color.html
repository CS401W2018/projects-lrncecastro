What is the difference between an ID and a Class?

  • ID follow a process such as you have to identify where you want to put the id in a tag, for example, <p id = "...">. And by putting #name in the css file you can change color etc. of what you are working on.
  • Class can be used in wide variety. It the same with the ID you have to put the class inside of the tag, for example, <li class = "blue">, there you can work on that specific part of it, but remember that in css you have to use .name of your class and the one that you want to do with it.
