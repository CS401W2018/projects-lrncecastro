Explain the difference between inline, embedded, and external CSS.

Inline CSS is the one we apply directly with the tag, and this only applies in that specific tag. For example if we use the <p> we have to put something inside, such as style = ".."

Embedded CSS is the one that has a style attribute but it is outside of the tag, rather we create something outside of the tag. For example we will use <style> </style> and inside of it we put our code for instance p for the paragraph followed by the { color = "blue" }

External CSS is the one that we use to access the other file that we created in the same folder that we have but this one has the extension of .css we use <link rel="stylesheet" href="name of the file"> so we can combine them in one html.

Explain why you should only use external CSS.

From what I understood because sometimes employer hires different kinds of programmer who focuses on just doing the website and the other is just focusing on doing the design of the website. Also for me I think it's easier to make edits if we use external CSS.

What is a selector used for?

We use selector so we can focus on something we are working on. For example I can change the color of the font of this specific paragraph to black because I inserted the style attribute inside and it is only selecting this part of the paragraph.
