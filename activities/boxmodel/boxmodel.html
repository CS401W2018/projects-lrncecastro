Box Model - Questions

1. Block tags, take up full width available, and it will start a new line. Examples are, <div>, <p>. Inline tags do not start on a new line, and only take the necessary width or height. Example is <span>
2. Margin is the side area of the background color of the where you put it. Border is the border of the tag where you put it, this can be dotted, thin solid, solid, dashed. Padding is the height of the background color, or sometimes it controls the width as well by using 4 different numbers inside it.
3. Units of measurement that you can use are cm, em, %, px
4. I would use the <figure> if I am working on an image that I have to caption on it. And I would use this tag i want to contain that part of website.
