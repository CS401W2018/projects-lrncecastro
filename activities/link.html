Explain the difference between an absolute link and a relative link. Why would you use relative links?

Absolute Link is the copy paste of the original link from another website.In this link it includes the original domain name and path. Relative Link is a link that points to a source relatively to the current page's location. For example: a link that brings you to the next page of the website or a link that shows you something like opening an image.
