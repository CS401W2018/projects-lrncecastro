When do you use a table?

When writing a data or information.

When do you not use a table?

To layout elements on the page.

When do you need to do extra work for table accessibility?

By changing the "td" to "th" and putting some other tags such as colspan and rowspan and headers will our table become accessible, and with that you have to put extra work by doing it.

Why don't we use the border attribute?

Because it is more powerful to use CSS to add border, because not only the border you can edit or adjust but can also put some colors and change the thickness of the border.

What is the easiest way to color every other row a different color?

There are ample of ways to color each row, you can use the parent of the row, or make a class or id of the row and edit them in CSS, or the best way is to select the parent class tag and call the "tr" so every "tr" we are working on it and make sure to add the ":nth-child(odd/even)" to work on it..
