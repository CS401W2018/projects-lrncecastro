Questions

Why is the viewport meta tag important?

It is important so you can work around web page without changing a lot of things in the css. The viewport gives you an idead of what is the website going to look like if you add or remove something in your css or html.

How can you adjust the CSS for printing?

Create a printing css that is connected to the html and everything that you don't want to be included when they are print the page will be removed.

When would you want to adjust the CSS for printing?

Print css should be done before the publishment of the web page so there will be no more issues.
