Internet

Internet or sometimes called net is a worldwide system of interconnected of computer network and electronic devices. The internet transmits data in the form of packets across network of routers, switches, and cables, making it possible for people to connect in all parts of the world real-time.

Web

Web also known as WWW (World Wide Web). This is a collection of interconnected web pages that people can access over the internet. In short, web is a platform for accesssing information and stuff published in the internet via browser.

How are the Internet and Web different?

Internet is a global network connected to other network. Internet is made up of hardware such as cables, routers, and servers that enable devices to communicate real-time. Web is one of a services that runs on the internet. Web is a collection of websites and web pages that can be accessed through web browser.

ASCII and Unicode

ASCII is one encoding standard used to represent a text in computers. It was the first major character encoding standard. And with that ASCII has increased its limitations because now it only supports character used in English langauge and a limited set of symbols. Unicode was invented to address the short coming of ASCII by giving a much larger character set.

What is UTF-8

UTF-8 (Unicode Transformation Format - 8-bit) is a character encoding standard. This represent characters in computers and other devices.

URL

URL (Uniform Resource Locator) is a unique address that points the a specific location on the internet. URL usually consits of a protocol, domain, and path to a specific resource.

Domain Name

Domain name is the name used to identify the website or other internet resources. It is a part of the URL and works as an identifier through website on internet.

HTTP

HTTP (Hypertext Transfer Protocol) is a protocol used to transfer hypertext request and data on WWW. It is the foundation of data communication on the internet.

HTTPS

HTTPS (Hypertext Transfer Protocol Secure) is a secure version of HTTP. Using encryption to protect the confidentiality of data between the server and client.

IP Address

IP Address (Internet Protocol Address) is the string of numbers and/or letters assigned to every device which is connected to a network. IP Address allows devices to communicate over the internet.

DNS

DNS (Domain Name System) is a system that translates domain name into an IP Address which are used by browsers to load. DNS serves like a phonebook of the internet.

W3C Recommendations

W3C Recommendations are standards and guidelines published by the W3C (World Wide Web Consutium). The guidelines and recommendations in W3C help cultivate the development of the web and technology.

XML, XHTML, HTML, HTML5

XML (Extensible Markup Language) a flexible text format to structure, store and transport data.

XHTML (Extensible Hypertext Markup Language) is a stricter and more XML- compliant version of HTML. It makes sure that web pages are properly structured.

HTML (Hypertext Markup Language) is the standard markup language to create web pages.

HTML5 is the latest version of HTML, with its new features user can create more interactive websites. It gives improved support for multimedia creation and others.
