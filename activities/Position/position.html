What is normal flow in HTML?

Normal flow is that each block tag takes the entire space of the screen.

What does float do? Why do you need to clear the float?

Float can put the image or something from left to right or vice versa. And you can remove the float by making any block tag. For example the <p> tag and create a class inside of that tag use the "clear" and type "both"

What do absolute, fixed, and relative do?

Absolute position removes the flow on the screen or browser, where after the one you are working on will stay still on the position, different to the relative position.

Fixed position just like the word suggest the element that has this one will be fixed in that position.

Relative position allows you to work on different positions on the screen by adding an x of pixels in top, left, right and bottom, and by so doing the one you are working on can sometimes go overlapping the other things you have on the browser if browser size has been changed.

How can you make nice navigation buttons for mobile devices?

We can make a nice navigation buttons for mobile devices by changing where the navigation is, changing its color, making it look that the navigation changes into something when it's click.
