Why is the viewport meta tag important?

It is important so you can work around web page without changing a lot of things in the css. The viewport gives you an idead of what is the website going to look like if you add or remove something in your css or html.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

The one that is entered last, which is the "flex-direction: row;"

How do you change the CSS for a web page displayed on a phone?

Using the @media only screen and (max-width: ....) with this you can change the display size of the website.
